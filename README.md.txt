# Kamus

Aplikasi PWA “Kamus Hafalan” — tambah perkataan + elemen + dialog + hafalan.

## Cara Deploy di GitHub Pages

1. Buat repository di GitHub bernama `Kamus` (username anda: `Sayaz06`).  
2. Upload semua fail & folder di root:  
   - index.html  
   - style.css  
   - app.js  
   - manifest.json  
   - sw.js  
   - icon-192.png  
   - icon-512.png  
3. Di repository → Settings → Pages → Source: main branch / root → Save.  
4. Tunggu beberapa minit. URL: `https://Sayaz06.github.io/Kamus/`  
5. Buka di telefon → pilihan “Add to Home Screen” akan muncul.

## Gantikan Icon

Anda boleh ganti `icon-192.png` & `icon-512.png` dengan icon buku kecil anda sendiri.

## Fitur

- Tambah perkataan  
- Tambah elemen (termasuk Ayat Dialog)  
- Buang elemen / buang perkataan  
- Dialog chat kiri-kanan  
- Tick elemen selesai  
- Export / Import JSON untuk cross-platform  
- PWA boleh install & offline support
